# 🎓 SLIIT GPA Calculator

A modern, fast, and responsive GPA calculator built specifically for **SLIIT (Sri Lanka Institute of Information Technology)** students.

Calculate your Semester GPA (SGPA), Weighted GPA (WGPA), and estimate your degree classification with an intuitive and user-friendly interface.

🌐 **Live Demo:** https://sliitgpa.vercel.app/

---

## ✨ Features

- 📚 Calculate Semester GPA (SGPA)
- 🎯 Calculate Weighted GPA (WGPA)
- 🏆 Predict Degree Classification
- 📝 Support for multiple SLIIT faculties and degree programs
- 📱 Fully responsive design
- ⚡ Fast and lightweight
- 🎨 Clean and modern UI
- 🌙 Easy-to-use interface

---

## 🚀 Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## 🛠️ Installation

Clone the repository

```bash
git clone https://github.com/ravindudanthanarayana/gpa.git
```

Go into the project

```bash
cd gpa
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

---

## 📦 Build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```

---

## 🎯 Purpose

This project was created to help SLIIT students quickly and accurately calculate their academic performance without manually computing GPA values.

It provides a simple interface for tracking semester progress and understanding the expected degree classification based on current academic results.
